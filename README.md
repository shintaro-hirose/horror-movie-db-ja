## ホラー映画DB

TMDB APIを利用して、ホラー映画を中心に検索できます。

提供する情報

- ポスター
- バックドロップ画像
- 概要
- 制作年
- キーワード
- 予告動画
- キャスト
- 他のおすすめ作品

[デモ](https://horror-movie-db.web.app)

## 使用した技術

- フロントエンド: React, Redux, redux-thunk, Material-UI

- Rest API: TMDB API, thunk/axiosによる非同期取得

- デプロイ: Firebase Hosting

- ソースコード管理: Github
